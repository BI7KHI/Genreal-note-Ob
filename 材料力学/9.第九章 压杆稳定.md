# 9.3 其它支座条件下细长压杆的临界压力
对于其他支座条件下细长压杆，求临界压力有两种方法：$F_{cr} = \frac{\pi^2EI}{l^2}$
1. 从挠曲线微分方程入手
2. 比较变形曲线
**一端固定一端自由**
$$F_{cr} = \frac{\pi^2EI}{(2l)^2}$$

# 9.4 欧拉公式的适用范围 经验公式
## 临界压力
$$临界压力\quad F_{cr} = \frac{\pi^2EI}{(\mu l^2)}$$
$$临界应力\quad \sigma_{cr} = \frac{F_{cr}}{A} = \frac{\pi^2EI}{(\mu l)^2A}$$
$$惯性半径\quad i=\sqrt{ \frac{I}{A} }\qquad \sigma_{cr} = \frac{\pi^2E}{\left( \frac{\mu l}{i} \right)^2} = \frac{\pi^2E}{\lambda}$$
$$\sigma_{cr} = \frac{\pi^2E}{\lambda^2} \qquad \lambda = \frac{\mu l}{i}$$

# 9.6 提高压杆稳定性的措施
$$欧拉公式 \quad = \frac{\pi^2EI}{(\mu l)^2}$$
$F_{cr}$越大越稳定
* 减小压杆长度$l$
* 减小长度系数$\mu$（增强约束）
* 增大截面惯性矩$I$（合理选择截面形状）
* 增大弹性模量$E$（合理选择材料）
