# 3.3 纯剪切
### 一、薄壁圆筒扭转时的切应力
由q-q 截面以左的部分圆通的平衡方程$\sum M_{x} =0$得
$$M_{e} = 2\pi r \delta \cdot \tau \cdot r$$
$$\tau = \frac{M_{e}}{2\pi r^2 \delta}$$
### 二、剪切力互等定理
采用截面法将圆筒截开，横截面上分布有与截面平行的切应力，由于壁很薄，可以假设切应力沿壁厚均匀分布
![[Pasted image 20260105150008.png]]
由平衡方程$\sum M_{z} = 0$，得到
$$M_{e} = 2\pi r \delta \cdot \tau \cdot r$$
$$\tau = \frac{M_{e}}{2\pi r^2 \delta}$$
在相互垂直的两个平面上，切应力必然成对存在，且数值相等；两者都垂直于两个平面的交线，方向则共同指向或背离这一交线
$$\tau\delta dy = (\tau' \delta dx)dy$$
$$\tau = \tau'$$
### 三、切应变 剪切胡克定律
在切应力的作用下，单元体的直 角将发生微小的改变，这个改变量$\mathbf{\gamma}$称为切应变。
$$\gamma=\frac{r\varphi}{l}$$
当切应力不超过材料的剪切比例极限时，切应变$\gamma$与切应力$\tau$成正比，这个关系称为**剪切胡克定律。**
$G$——切变模量($GN / m^2$)，衡量材料抵抗剪切变形的**刚度**
$$\tau = G \gamma$$
# 3.4 圆轴扭转时的应力
### 1. 变形几何关系
圆轴扭转的平面假设
> 圆轴扭转变形前原为平面的横截面，变形后仍保持为平面，形状 和大小不变，半径仍保持为直线；且相邻两截面间的距离不变。

## 2. 物理关系
根据剪切胡克定律$\tau = G \gamma \quad \gamma_{p} = \frac{\rho d\varphi}{dx}$
距离圆心为$\rho$处的切应力$\tau_{\rho} = G \gamma_{\rho} = G\rho  \frac{d\varphi}{dx}$

## 3. 静力关系
$$T = \int_{A}\rho \tau_{\rho}dA = G \frac{d\varphi}{dx}\int_{A} \rho^2dA$$
$$\mathbf{\tau_{\rho} = \frac{T}{I_{p}}}\rho$$
上述公式适用于
1. 等直圆杆
2. $\tau_{max} \le \tau_{p}$

> 横截面上某点的切应力的方向与扭矩方向相 同，并垂直于半径。切应力的大小与其和圆 心的距离成正比。

$$\tau_{max} = \frac{TR}{I_{p}} = \frac{T}{\frac{I_{p}}{R}}$$

令$W_{t} = \frac{I_{p}}{R}$,则**抗扭截面系数**
$$\mathbf{W_{t} = \tau_{max} = \frac{I_{p}}{R}}$$
> $I_{p}$为截面的极惯性矩
> $R$为截面上距离中心最远点到轴线的距离(对圆轴即**外半径**)

> 扭转强度条件：$\tau_{max} = \frac{T_{max}}{W_{t}} \le [\tau]$

1. 校核强度   $\tau_{max} = \frac{T_{max}}{W_{t}} \le [\tau]$
2. 设计截面   $W_{t} \ge \frac{T_{max}}{[\tau]}$
3. 确定载荷   $T_{max} \le W_{t}[\tau]$

# 3.4 圆轴扭转时的变形

### 扭转强度条件
$$\tau = \frac{T}{W_{t}} \le [\tau]$$
$$W_{t} = \frac{1}{16}\pi D^3$$
* 已知$T、D和\tau$，**校核强度**
* 已知$T和[\tau]$ 设计截面
* 已知$D和[\tau]$，**确定许可载荷**
### 扭转刚度条件
$$ \varphi'_{max}= \frac{T}{GI_{p}} \le [\varphi']$$
$$I_{p} = \frac{1}{32}\pi D^4$$
* 已知$T、D和 [\varphi']$，**校核刚度**
* 已知$T和[\varphi']$,**设计截面**
* 已知$D和[\varphi']$**确定许可载荷**
