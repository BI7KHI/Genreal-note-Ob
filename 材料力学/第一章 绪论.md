### 平均应力$p_{m}$
$$
p_{m}= \frac{\Delta F}{\Delta A}
$$
$p_{m}$是一个矢量，代表在$\Delta A$范围内，单位面积上内力的平均集度，称为平均应力。当$\Delta A$趋于0时，$p_{m}$的大小和方向都将趋于一定极限
$$
p=\lim_{ \Delta A \to 0 }P_{m} = \lim_{ \Delta A \to 0 } \frac{\Delta F}{\Delta A} 
$$
$p$称为C点的应力。它是分布内力系在C点的集度，反映力系在C点的强弱程度。$p$s是一个矢量，一般说既不与界面垂直，也不与界面相切。
通常把应力p分解为垂直于截面的分量$\sigma$和切于界面的分量$\tau$,$\sigma$称为正应力，$\tau称为$切应力
### 单位
应力的单位为Pa，1Pa = $1N$/$m^2$，工程实际中使用不便，通常使用MPa，1Mpa = $10^6$Pa
# 1.5 变形与形变

## 正应变
$$
\varepsilon  = \lim_{ MN \to 0 } \frac{M'N'-MN}{MN}=\lim_{ \Delta x \to 0 } \frac{\Delta s}{\Delta x}
$$
$\varepsilon$称M点沿x方向的线应变称为正应变
### 切应变
$$
\gamma = \lim_{ ML \to 0 ,ML \to 0} \left( \frac{\pi}{2}-\angle L'M'N' \right)
$$
