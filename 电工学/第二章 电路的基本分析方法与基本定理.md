# 2.1 电路的等效变换
## 2.1.1 等效变换的概念
1. 等效只是针对网络的端口而言，对外电路等效，对内电路并不一定等效
2. 在等效条件下，两个互为等效的网络之间可以相互替换
3. 对于多网格端口来说，如果外部各对应端口之间的伏安关系完全相同
## 2.1.2 电阻电路的等效变换
### 电阻的串联
当n个电电阻串联时，可用一个等效电阻R来等值代替，其值等于这n个串联的电阻之和。
$$R ={\textstyle \sum_{n}^{1}}R_{i}$$
### 电阻的并联
如果电路中有两个或更多个电阻联接在两个公共的结点之间，则这样的联接法就称为电阻的并联。
![[Pasted image 20251218234328.png]]
个电阻并联时，可用等效电阻R代替，等效电阻R的倒数，等于各并联电阻倒数之和
$$\frac{1}{r} = \frac{1}{R_{1}}+ \frac{1}{R_{2}} +\dots + \frac{1}{R_{n}}\quad 或\quad G = G_1 + G_{2} + \dots +G_{n}$$
$G$称为电导，单位名称西门子(S)
#### 并联电路的特点
1. 电阻两端电压相等
2. $I = \sum I_{n}$
3. 并联电阻的数目越多，等效电阻R越小，如果端电压保持不变，则总电流越大。
> 阻值相差很大的两个电阻并联，大电阻的分流作用常可忽略不计。
> 阻值相差很大的两个电阻串联，小电阻的分压作用常可忽略不计；

# 2.5 叠加定理
> 将多电源复杂电路化为几个单电源电路来计算

>在有多个独立电源同时作用的线性电路中，任意支路的电流或任意两点间的电压，等于各个电源单独作用时所得结果的代数和。

* 一个电源单独作用时，其它独立源失效
* 独立源失效意为
	*  **独立电压源短路**
	*  **独立电流源开路**

### 应用叠加定理要注意的问题
1. 叠加定理只适用于线性电路（电路参数不随电压、电流的变化而改变）。
2. 叠加时只将电源分别考虑，电路的结构和参数不变。
暂时不予考虑的 **恒压源应予以短路**，即令U=0；
暂时不予考虑的 **恒流源应予以开路**，即令 Is=0。
3. 解题时要标明各支路电流、电压的**参考方向**，而且分电路的各支路电流与电压的参考方向保持与原电路一致。原电路中各电压、电流的最后结果是各分电压、分电流的代数和。
4. 叠加原理只能用于电压或电流的计算，不能用来求功率，即功率不能叠加。
