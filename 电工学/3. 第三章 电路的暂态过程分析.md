# 3.1 RLC元件的物理特性
## 3.1.1 电阻元件
**电阻的定义**
$$R = \frac{u(t)}{i(t)}$$
* 对于直流电阻和低频电阻而言，R通常是一个恒定的值
* 此时R只与电阻本身的**材料、结构、尺寸**有关

* 对于**RF射频**电阻而言，由于趋肤效应，R还与频率有关(特性阻抗)

**电阻的能量**
$$W = \int_{0}^{t}uidt = \int_{0}^{t}Ri^2dt \ge 0$$
表明电能全部消耗在电阻上，转换为热能散发
## 3.1.2 电容元件
### 伏库特性
$$q = Cu$$
$$C = \frac{q}{u} = \tan \alpha$$
$C = \frac{\varepsilon S}{d}$（$\varepsilon$为极板间介质的介电常数，S为极板面积，d为极板间距）
### 电容元件的电压、电流关系
$$i = \frac{dq}{dt} = C \frac{du}{dt}$$
* i的大小取决于u的变化率，与u的大小无关——电容是动态元件
* u当u为常数（直流电压）时，i=0。电容相当于开路——电容有隔断直流电流的作用
* u实际电路中通过电容的电流i是有限值，则电容电压u必定是时间的连续函数——电容两端的电压不能突变。
$$i = C \frac{du}{dt} = C \lim_{ \Delta t \to 0 } \frac{\Delta u}{\Delta t} = C \frac{u_{2}-u_{1}}{t_{0}- t_{0}} \to \infty $$
### 电容的功率和储能
$$i = C \frac{du}{dt}$$
$$p = ui = uc \frac{du}{dt}$$
>电容能在一段时间内吸收外部供给的能量，转换为电场能量储存起来，在另一段时间内它又可以把能量释放回电路，因此电容元件是储能元件，它本身不消耗能量。

1. 电容的储能只与当时的电压值有关，电容电压不能越变——储能不能越变
2. 电容存储的能量一定$\ge$ 0
## 3.1.3 电感元件
当电流通过**单匝**线圈时，将产生磁通$\mathbf{\phi}$ 单位：韦伯（Wb）
N匝线圈产生的总磁通——磁通链$\psi$
$$\psi = N\phi$$
### 电感元件的积分式
$$u(t) = L \frac{di(t)}{dt}$$
$$i(t) = \frac{1}{L}\int_{-\infty}^{t}  \, udt  = \frac{1}{L}\int_{-\infty}^{t_{0}}  \, udt  + \frac{1}{L} \int_{t}^{t_{0}}  \, udt  = i(t_{0}) + \frac{1}{L}\int_{t_{0}}^{t} udt$$
* $t_{0}$研究问题的起始状态
* $i(t_{0})$——电感电流的初始值，称为初始状态， 它反映电感初始时刻的储能状态。
### 电感的功率和储能
$$u(t) = L \frac{di(t)}{dt}$$
$$\mathbf{功率} \quad p = ui = iL \frac{di}{dt}$$
> 电感能在一段时间内吸收外部供给的能量，转换为磁场能量储存起来，在另一段时间内它又把能量释放回电路，因此电感元件是储能元件，它本身不消耗能量

若 $i(\infty)=0$
则$W_L= \frac{1}{2} Li^2(t) \ge 0$
1. 电感的储能只与当时的电流值有关，电感电流不能跃变——电感储能不能跃变
2. 电感储存的能量一定≥0

## 3.1.3 电感元件与电容元件的比较
|     | 电容$C$                                                                            | 电感$L$                                                                                     |
| --- | -------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| 变量  | 电压$\quad u$<br>电荷$\quad q$                                                       | 电流$\quad i$<br>磁链$\quad \psi$                                                             |
| 关系式 | $q = Cu$<br>$i = C \frac{du}{dt}$<br>$W_{C} = \frac{1}{2}Cu^2 = \frac{1}{2C}q^2$ | $\psi = Li$<br>$u = L \frac{di}{dt}$<br>$W_{L}   = \frac{1}{2} Li^2 = \frac{1}{2L}\psi^2$ |
**结论**
1. 元件方程的形式是相似的
2. 若把$u-i , q-\psi C-L$互换，可由电容元件的方程得到电感元件的方程

# 3.2 换路定则
## 3.2.1 换路定则
  换路：电路状态的改变，如：
  1. 电路接通，断开电源
  2. 电路中电源的升高和降低
  3. 电路中元件参数的改变
![[Pasted image 20260115092725.png]] 
### 小结
1. 换路瞬间，$u_{c},i_{L}$不能突变，其它电量均可能突变，变不变由计算结果决定
2. 在稳态电路里，电容相当于开路，电感相当于短路
3. 换路瞬间，若$u_{C}(0^-) = 0$，电容相当于短路
   若$u_{c}(0^-) = u_{0} \ne 0$,电容相当于恒压源
4. 换路瞬间，若$i_{L}(0^-) = 0$电感相当于断路
   若$i_{L}(0^-) = I_{0} \ne 0$电感相当于恒流源

# 3.3 RC电路的响应
## 一阶电路
* 根据电路规律列写电压、电流的微分方程，若微分方程是一阶的，则该电路为一阶电路
* u只含有一个储能元件或者可等效为一个储能元件的线性电路，它的微分方程都是一阶线性微分方程——一阶线性电路
## 3.3.2一阶RC电路的**零输入响应**
> RC电路无电源驱动，输入信号为零。在此条件下，由电容器的初始状态uC(0+)所产生的电路响应——RC电路零输入响应

## 3.3.3 一阶线性RC电路的全响应
> RC电路有电源驱动，而且电容元件的初始状态uC(0+)不为零时的电路响应——RC电路全响应

线性电路：RC电路全响应 = RC电路零状态响应 + RC电路零输入响应
